AquaFlow

Este es un proyecto para la carrera de Ingeniería en Mecatrónica para la Universidad Invenio, sede Heredia, Costa Rica.

Esta web se realiza con la afinidad de mostrar de manera virtual el funcionamiento aproximado del proyecto en gestión, 
su función principal es lograr una correcta purificación del agua independientemente de su origen sea este un rió, lluvia, entre otras.

El objetivo es demostrar el como con los materiales adecuados, este sistema de purificación es una 
opción la cual es viable para poblaciones a nivel nacional como internacional y al mismo tiempo no presenta un gasto muy alto de dinero.

La explicación de su funcionamiento esta disponible en la web: https://imoya03.github.io/AquaFlow/

La lista de materiales asimismo como los archivos de impresión en 3D se encuentran disponibles en la misma web como en este repositorio

Cualquier duda la pueden realizar al correo: imoyaa@student.invenio.ac.cr


AquaFlow
This is a project for the Mechatronics Engineering career for the Invenio University, Heredia campus, Costa Rica.

This website is made with the affinity of showing virtually the approximate operation of the project under management, 
Its main function is to achieve a correct purification of water regardless of its origin, whether it is a river, rain, among others.

The objective is to demonstrate how with the right materials, this purification system is a 
an option which is viable for populations at the national and international level and at the same time does not present a very high expenditure of money.

The explanation of how it works is available on the website: https://imoya03.github.io/AquaFlow/

The bill of materials as well as the 3D printing files are available on the same website as in this repository

Any questions can be made to the email: imoyaa@student.invenio.ac.cr
